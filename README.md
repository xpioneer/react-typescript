#react-typescript

### 项目概况
***
使用蚂蚁金服的Antd开发的一套后台管理系统，主要用到了react、typescript、antd等技术，使用webpack4打包构建

### 项目主要技术结构

***
* react
* typescript
* antd
* mobx
* webpack4
* react-router4

### 安装
***
在终端下操作

项目地址: (`git clone`)

```
git clone git@github.com:xpioneer/react-typescript.git
```

安装node_modules依赖 `yarn`

```
yarn #in your command terminal
```
***


### 运行
启动开发服务: (http://localhost:8022)

```
yarn start
```

生产环境打包

```
yarn build
```
